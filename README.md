## Задание 1. Кеширование

Кеширование помогает, когда:
1 База данных или сервис перегружены частыми одинаковыми запросами.
2 Страницы отчёты долго считаются (можно сохранить готовый результат и отдавать быстрее).
3 Есть внешние API с ограничениями или высокой задержкой (кешируем ответы и реже дергаем API).
4 Много однотипных данных (например, список товаров и категорий)  можно быстро отдавать из кеша.
5 Нужно уменьшить задержки для пользователя (быстрее открываются страницы/приложение).
